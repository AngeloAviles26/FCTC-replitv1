# Project: From Campus to Career (Mobile Frontend)

## Overview
An AI-powered skill gap analysis and career readiness system for Filipino university students.

## Target Audience
* Filipino BSIT/BSCS students
* Mobile-first users (Android focus)
* Mid-range devices, mobile data users

## Tech Stack
* React Native (Expo)
* NativeWind (Tailwind CSS)
* React Navigation
* Zustand (State Management)
* Axios (API)
* Lucide-React-Native (Icons)

## UI/UX Style
* Clean, modern, minimal
* Primary Color: RED (Filipino academic/national association)
* Card-based layouts (rounded-2xl)
* Responsive for mobile-first

## Screens & Features

### 1. Authentication
* **Login:** Email/Password
* **Register:** Name, Email, Password, Program, Year Level

### 2. Dashboard (Home)
* Readiness Score (Progress bar/circular)
* Target Career (e.g., Data Analyst)
* Skill Match Summary
* Key Metrics Cards

### 3. Gap Analysis
* List of skill gaps
* SDI (Demand %)
* Priority Tags: Critical, Important, Emerging
* Detail view interaction

### 4. Roadmap
* Tabs: Certifications, Projects
* Items: Title, Skills, Status
* Progress tracking buttons

### 5. Skill Decay Warning
* Declining skills alerts
* Trend indicators (down arrows)
* Replacement skill suggestions
* High-visibility alerts

### 6. Profile
* Student info
* Skills/Certifications/Projects lists
* Edit functionality

## Navigation
* Bottom Tabs: Home, Gap Analysis, Roadmap, Profile
